## What This Is

GTC 2026 runs March 16-19, 2026, in San Jose + virtual. Over 700 sessions.
Jensen Huang keynote. This is where NVIDIA announces new CUDA features,
library improvements, and architecture details.

## Why It Matters for Us

GTC is historically where NVIDIA reveals:
- New cuBLAS/cuSOLVER performance improvements
- CUTLASS updates with new kernel architectures
- CUDA toolkit feature announcements (PTX ISA extensions, new instructions)
- Architecture deep-dives (Blackwell tensor core details we haven't seen)
- Kernel optimization case studies from NVIDIA engineers

## Sessions to Watch

The session catalog should be checked for talks covering:
1. **Blackwell kernel optimization** -- any sm_120/sm_100 kernel tuning talks
2. **CUTLASS updates** -- new EVT capabilities, Blackwell examples
3. **cuBLAS/cuSOLVER** -- library improvements for Blackwell
4. **Kernel fusion frameworks** -- TensorRT-LLM internals, FlashInfer updates
5. **Mixed precision** -- FP8/NVFP4 kernel design patterns
6. **Linear algebra on GPU** -- TRSM, factorization, batched solvers
